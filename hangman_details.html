Overview

A command-line based Hangman game developed in C++ to demonstrate proficiency in memory management and object-oriented design. The project focuses on the fundamental principles of C++, including pointer manipulation, array handling, and custom class structures to manage game states and string comparisons efficiently.

Memory Optimization

Direct pointer manipulation for efficient handling of character arrays.

Low-Level Control

Manual management of game states using core C++ libraries.

Robust Error Handling

Comprehensive logic to manage edge cases and user input exceptions.

Takeaways

Developing Hangman in C++ was a foundational exercise in managing data at a lower level. I focused on pointer manipulation to efficiently navigate and modify character arrays, which deepened my understanding of how software interacts directly with memory. The project required a disciplined development methodology, ensuring that exception handling and edge cases were managed through robust logic and custom class structures.

Technologies Used
  • C++ (Standard Library)
  • Pointer & Memory Control
  • Array & String Manipulation
  • Exception Handling & Testing
