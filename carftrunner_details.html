Overview

A native Android game application developed to explore the Android SDK and mobile-specific logic. Built using Java and XML for layout design, the project focuses on real-time event handling and the application lifecycle. By leveraging the Gradle build system, I ensured a structured development workflow while implementing a functional prototype that demonstrates core Android engineering principles.

Real-Time Logic

Smooth event handling and game-state management in a native Java environment.

Dynamic UI

Responsive layouts built with XML to adapt to different screen densities.

Lifecycle Management

Efficient resource handling to maintain performance during active gameplay.

Takeaways

Developing CraftRunner provided practical insights into the Android Activity Lifecycle and native resource management. The core challenge was synchronizing real-time game logic with responsive UI updates—an exercise that strengthened my ability to build well-structured solutions in Java. This project gave me hands-on experience in navigating the Android SDK and understanding the engineering principles required to maintain a stable and engaging mobile experience.

Technologies Used
  • Java (Android Framework)
  • XML UI Design
  • Gradle Build System
  • Activity Lifecycle Management
